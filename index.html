WebAPI Token Auth Bootstrap

Out of the box Token based User Auth for WebAPI applications, Provides ready to use 'TokenAuthorize' Attribute and 'TokenAuthApiController' Controller.

Download this project as a .zip file Download this project as a tar.gz file

Token Based User Authentication

User Property inside the TokenAuthApiController (Id, Username, Role, LastAccess).

Token Based User Authorization

TokenAuthorizeAttribute with Access Level - Public, User, Admin or Anonymous.

Built-in Functionality

Login(), Logoff(), Error(), Unauthorized() Responses with various overloads.

Shared Dynamic Object Betwen Client and Server

UserData Property inside the TokenAuthApiController (Up to size of 4 KB storage). Great for caching data in the client side, especially for thin back-end applications like Single Page Applications.
